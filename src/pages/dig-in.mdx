# Dig In To Web Accessibility

Keep digging deeper! Here are an assortment of resources, articles and courses -- free, paid, and with subscription.

So when are you an accessibility professional? My answer is... _shrug_? Keep caring, keep learning, keep championing change in your teams and codebases. To read more on this topic, check out [Accessibility Pro Certified: To Be or Not To Be](https://www.24a11y.com/2018/accessibility-pro-certified/).

## Courses

### Free

- [Web Accessibility](https://www.udacity.com/course/web-accessibility--ud891): A free Udacity course created as a partnership between Google and AT&T.

### With Subscription

- [Accessibility in JavaScript Applications](https://frontendmasters.com/courses/javascript-accessibility/): A Frontend Masters course created by [Marcy Sutton](https://twitter.com/marcysutton).
- [Start Building Accessible Web Applications Today](https://egghead.io/courses/start-building-accessible-web-applications-today): An Egghead course created by [Marcy Sutton](https://twitter.com/marcysutton) introducing accessibility through hands-on, technical examples.
- [Hacking Digital Styleguides for Accessibility: Type, Color, Imagery](https://www.skillshare.com/classes/Hacking-Digital-Styleguides-for-Accessibility-Type-Color-Imagery/1920202818): A Skillshare course created by [Tatiana Mac](https://twitter.com/tatianatmac), focusing on the basics of accessibility with a design lens.

### Paid

- [Deque University](https://dequeuniversity.com/curriculum/online-classes/): [Deque](https://www.deque.com/) is a digital accessibility company offering curriculum through their Deque Unniversity platform. Maybe useful in particular if you're specifically interested in becoming [certified via the International Association of Accessibility Professionals](https://www.accessibilityassociation.org/certification).

## Intro Guides

- [WebAIM Introduction to Web Accessibility](https://webaim.org/intro/): A web accessibility introduction provided by WebAIM.
- [Mozilla Accessibility Guide](https://developer.mozilla.org/en-US/docs/Learn/Accessibility): A learning module on accessbility comprised of multiple guides, provided by Mozilla.

## Resources

- [The A11y Project](https://www.a11yproject.com/) is an open source accessibility resource site.
- [A11y Weekly](https://a11yweekly.com/) is a weekly newsletter with web accessibility content and news, curated by [David Kennedy](https://twitter.com/DavidAKennedy).

## Articles

- [24 Accessibility](https://www.24a11y.com/) is a series of articles related to web accessibility. New content is published every day for the first 24 days of December each year.
- [A List Apart (Topic: Accessibility)](https://alistapart.com/blog/topic/accessibility/). A List Apart is a web zine about web topics. These are all the articles published on A List Apart tagged with `accessibility`.

## Communities

- [Web A11y Slack](https://web-a11y.herokuapp.com/) is a Slack team devoted to web accessibility topical discussion. Ask questions, share resources, get advice, etc.
